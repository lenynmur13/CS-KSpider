================================================================================
                         STUDY METHODS SUPPORTED
================================================================================

1. FLASHCARDS (Q/A) ✅
   - Question on front, answer on back
   - Tags for categorization
   - Difficulty levels (easy, medium, hard)
   - Export to Anki CSV for spaced repetition

2. POP QUIZ (Multiple Choice) ✅
   - 4 answer options (A, B, C, D)
   - Correct answer indication
   - Explanations for each question

3. PRACTICE TEST (Mixed Types) ✅
   - Multiple choice questions
   - True/False questions
   - Short answer questions
   - Point values and total scoring
   - Comprehensive answer key

4. AUDIO SUMMARY (TTS) ✅
   - Text-based summary generation (TTS-friendly format)
   - Key concepts with importance levels
   - Main takeaways as complete sentences
   - Overview and conclusion sections
   - Estimated read-aloud time calculation

================================================================================
                         INPUT FORMATS SUPPORTED
================================================================================

DOCUMENTS:
- .pptx - PowerPoint presentations (slides, notes, tables)
- .pdf  - PDF documents (all pages)
- .txt  - Plain text files
- .md   - Markdown files (with title extraction)

VIDEO/AUDIO:
- .mp4, .mov, .webm, .avi, .mkv - Video files
- .mp3, .wav, .m4a, .aac, .ogg  - Audio files
- Automatic audio extraction from video
- Chunked transcription for large files (>25MB limit)

================================================================================
                         OUTPUT FORMATS SUPPORTED
================================================================================

- JSON: Canonical format with full metadata
- Anki CSV: Semicolon-separated for direct Anki import
- Markdown: Human-readable formatted documents

================================================================================
                         COST GUARDRAILS
================================================================================

1. AUDIO TRANSCRIPTION:
   - Audio chunked to ≤20MB segments
   - Converted to mono 16kHz 64kbps MP3
   - Reduces file size and API costs

2. TEXT GENERATION:
   - Max 5 chunks per generation (configurable)
   - Max 4000 tokens per response (configurable)
   - Token usage tracked per study set

3. CONFIGURABLE LIMITS:
   - STUDY_GUIDE_MAX_CHUNKS_PER_GENERATION
   - STUDY_GUIDE_MAX_TOKENS_PER_RESPONSE
   - STUDY_GUIDE_AUDIO_CHUNK_SIZE_MB

================================================================================
                         CONFIGURATION
================================================================================

Required:
  OPENAI_API_KEY           - OpenAI API key for generation and transcription

Optional:
  STUDY_GUIDE_DB_PATH      - Database file path (default: ./data/study_guide.db)
  STUDY_GUIDE_EXPORT_DIR   - Export directory (default: ./data/exports)
  STUDY_GUIDE_GENERATION_MODEL      - GPT model (default: gpt-4o)
  STUDY_GUIDE_TRANSCRIPTION_MODEL   - Whisper model (default: whisper-1)
  STUDY_GUIDE_MAX_CHUNKS_PER_GENERATION  - Chunks limit (default: 5)
  STUDY_GUIDE_MAX_TOKENS_PER_RESPONSE    - Token limit (default: 4000)
  STUDY_GUIDE_AUDIO_CHUNK_SIZE_MB        - Audio chunk size (default: 20)

================================================================================
                         CLI REFERENCE
================================================================================

# Initialize database
python -m study_guide init

# Show status
python -m study_guide status

# Ingest files
python -m study_guide ingest ./materials
python -m study_guide ingest ./materials --skip-existing

# List content
python -m study_guide list documents [--limit N]
python -m study_guide list chunks --doc ID [--limit N]
python -m study_guide list sets [--limit N]

# Generate study materials
python -m study_guide generate flashcards --doc ID [--count N]
python -m study_guide generate quiz --doc ID [--count N]
python -m study_guide generate test --doc ID [--count N]
python -m study_guide generate summary --doc ID [--points N]

# Export study sets
python -m study_guide export SET_ID --format json
python -m study_guide export SET_ID --format anki
python -m study_guide export SET_ID --format markdown
python -m study_guide export SET_ID -o ./custom/path.json

# Show study set content
python -m study_guide show SET_ID
